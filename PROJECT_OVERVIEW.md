### What is this?
An interactive Python tool for generating biomimetic bone scaffolds with gradient porous structures using 3D Voronoi tessellation. Designed for tissue engineering and regenerative medicine research.

### Key Features
- **Interactive Interface**: Direct numerical input with real-time visualization
- **Biomimetic Design**: Gradient structure mimicking natural bone (cortical â†’ transition â†’ trabecular)
- **3D Voronoi Tessellation**: Realistic porous architecture generation
- **Multiple Visualizations**: Colorful 3D models and SEM-style renderings
- **3D Printing Ready**: STL export for direct fabrication

### Who is this for?
- Tissue engineers
- Biomaterials researchers
- 3D printing specialists
- Stem cell researchers
- Anyone interested in computational biomimicry

### Tech Stack
- Python 3.7+
- NumPy & SciPy (scientific computing)
- Matplotlib (3D visualization)
- numpy-stl (STL export)
