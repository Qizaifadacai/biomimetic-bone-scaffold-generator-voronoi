# 快速参考卡片 - 新版本

## 🎮 启动命令

```bash
python 新版本演示.py
```
或
```bash
python 支持梯度的Voronoi支架生成器.py
# 选择: 1
```

## ⌨️ 输入框交互

```
1. 点击输入框
2. 输入数值（例如: 30000）
3. 按 Enter 确认
4. 点击 "Generate Scaffold"
```

## 🎨 生成的可视化

### 点击 "Save Visuals" 后自动生成：

| 文件 | 内容 | 特点 |
|------|------|------|
| `colorful_voronoi_3d_*.png` | 彩色Voronoi | 30+种颜色，多面体 |
| `realistic_scaffold_*.png` | SEM仿真支架 | 灰度，4视角 |
| `gradient_analysis_*.png` | 梯度分析 | 8个统计图表 |

## 🔘 按钮功能

| 按钮 | 快捷键 | 功能 |
|------|-------|------|
| 🟢 Generate Scaffold | - | 生成新支架 |
| 🔵 Save STL | - | 保存STL+JSON |
| 🟡 Save Visuals | - | 保存3张图片 |

## 💡 专业提示

1. **参数调试**：先用小密度值快速预览，满意后增大
2. **批量修改**：所有参数改完再点Generate（节省时间）
3. **高清图片**：使用Save Visuals按钮（自动300 DPI）
4. **时间戳**：文件名自动加时间戳，不会覆盖
